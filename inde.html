Checkbox Topic Specific Topics to Cover
HTML5 HTML elements, attributes, semantic HTML, forms, multimedia elements (audio, video, canvas),
CSS3 CSS selectors, specificity, box model, flexbox, grid layout, transitions, animations, media queries, preprocessors (SASS, LESS)
JavaScript Variables, data types, operators, conditionals, loops, functions, ES6 features (let, const, arrow functions, promises, async/await), DOM manipulation, event handling, JSON, basics of jQuery, introduction to modern frameworks (React, Vue.js, AngularJS)
React.Js Components, JSX, state, props, lifecycle methods, hooks (useState, useEffect, useContext, custom hooks), advanced topics (Context API, Redux, React Router)
Node.js & Express.js Basics of Node.js, modules, npm, creating a server, working with the filesystem, setting up an Express server, routing, middleware, RESTful APIs, error handling, authentication (JWT, OAuth)
Databases SQL : (basics, CRUD operations, joins, transactions, indexing); NoSQL : (basics of MongoDB, CRUD operations, schema design), ORMs (Sequelize, Mongoose)
Git & GitHub Basic Git commands (init, clone, commit, push, pull, branch), collaboration (forking, cloning, pull requests, merging), branching strategies (Git flow, feature branches)
DevOps Basics CI/CD (basics, setting up pipelines, tools like Jenkins, GitHub Actions), deployment (deploying applications to platforms like Heroku, AWS), cloud services (basics of AWS, Azure)
